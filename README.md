## 🚀 Getting Started

### 1️⃣ Clone the repository

```bash
git clone https://github.com/harshpal-18/your-portfolio.git
```

### 2️⃣ Navigate into the project

```bash
cd your-portfolio
```

### 3️⃣ Install dependencies

```bash
npm install
```

### 4️⃣ Start development server

```bash
npm run dev
```
